# GitHub: Explorando todas sus funciones para CPESGI

Este repositorio tiene como objetivo aprender y demostrar el uso de las funciones esenciales de GitHub, aplicado al trabajo dentro de CPESGI.

## 1. ¿Qué es GitHub?
GitHub es una plataforma basada en Git que permite almacenar, dar seguimiento y colaborar en proyectos.  

## 2. Repositorios
Este repositorio es el ejemplo: **github-cpesgi-exploracion**  
Un repositorio es un espacio donde vive un proyecto.

## 3. Commits
Son “fotografías” del progreso.  
Cada cambio debe incluir un mensaje claro explicando qué se modificó.

## 4. Branches (Ramas)
Las ramas permiten trabajar sin afectar la versión principal del proyecto.  
Crearemos una rama para agregar código en Python.

## 5. Pull Requests
Sirven para solicitar que una rama se fusione con la principal y para revisar cambios.

## 6. Clone y Pull
Estos comandos permiten trabajar desde tu computadora local

## 7. Issues
Se utilizan para registrar tareas, problemas o mejoras.

## 8. Markdown
Este archivo está escrito en Markdown, un formato simple para dar estilo al texto.
